# 就医前准备与就医后记录

帮助用户充分准备就医，基于健康档案生成个性化的就医清单，并规范就医后的记录更新。

## 核心任务

### 就医前准备
基于用户的健康档案，生成针对性的就医准备材料，提高就医效率。

### 就医后记录
标准化记录就医结果，更新健康档案。

## 就医前准备内容

### 1. 历史摘要（1页纸格式）

**相关既往史**
- 与本次就医相关的历史疾病
- 手术史、住院史
- 家族病史（如相关）

**相关用药**
- 当前正在服用的药物
- 过敏史和不良反应史
- 近期停用的相关药物

**相关检查结果和趋势**
- 最近2-3次相关检查的数值对比
- 异常指标的变化趋势
- 之前医生的诊断和建议

### 2. 本次重点关注指标

基于历史数据分析：
- 需要重点检查的项目
- 建议的检查频率
- 与症状相关的可能检查

### 3. 建议追问医生的问题（5-8个）

**个性化问题**（非泛泛提问）：
- 基于个人病史的具体疑问
- 关于检查结果的深入解读
- 治疗方案的个性化调整
- 复查时间和指标的确认

**示例**：
❌ 泛泛问题："这个病严重吗？"
✅ 个性化问题："我的血糖从5.2升到6.8，考虑我的家族糖尿病史，现在需要开始用药吗？"

### 4. 就医后记录模板

预设结构化模板，方便快速记录：

```
## 就医记录 - [日期] [科室]

### 医生诊断
-

### 检查结果
-

### 治疗方案
-

### 用药调整
-

### 复查计划
-

### 医生特别提醒
-

### 我的疑问（未问或需要下次确认）
-
```

## 就医后处理流程

当用户提供就医结果时，自动执行：

### 1. 更新就医记录
- 标准化记录医生的诊断和建议
- 更新用药清单
- 记录新的复查计划

### 2. 更新体检档案
- 将新的检查结果加入趋势分析
- 更新相关健康指标
- 调整健康风险评估

### 3. 复查提醒设置
- 将复查计划转换为具体日期
- 设置提前提醒时间
- 加入复查日历

## 专科特化

### 内分泌科
- 重点关注：血糖、甲状腺功能、代谢综合征指标
- 常见问题：药物调整、饮食控制、监测频率

### 心血管科
- 重点关注：血压、心电图、心脏超声、血脂
- 常见问题：用药时间、运动限制、危险信号

### 消化科
- 重点关注：肝功能、消化道症状、既往检查
- 常见问题：饮食禁忌、药物对消化系统影响

- 用户说要去看某个科室
- 用户询问"看医生要准备什么"
- 用户看完病回来，要更新记录
- 用户对就医结果有疑问，需要整理
