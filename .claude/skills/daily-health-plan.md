# 个性化日常健康管理方案

基于个人健康档案生成定制化的日常健康管理清单，涵盖饮食、运动、用药、监测等各方面。

## 核心任务

将复杂的健康数据和医疗建议转换为简洁、可执行的日常管理方案，帮助用户建立健康的生活习惯。

## 分析基础

### 数据输入
- 体检档案（血糖、血脂、肝肾功能等）
- 既往病史和手术史
- 当前用药清单
- 运动监测数据（Apple Watch 等）
- 家族病史和遗传风险

### 个性化因子
- 年龄和性别
- 职业和生活方式
- 地理位置和气候
- 经济条件和就医便利性

## 管理方案结构

### 🥗 饮食管理

**基于血糖水平**
- 血糖正常：均衡饮食，控制精制糖
- 血糖偏高：限制碳水化合物，增加膳食纤维
- 糖尿病：严格血糖指数管理

**基于血脂水平**
- 胆固醇高：减少饱和脂肪，增加不饱和脂肪
- 甘油三酯高：控制总热量，减少精制糖

**具体食物建议**
- ✅ 推荐：燕麦、深绿叶蔬菜、三文鱼、坚果
- ⚠️ 限制：白米饭、甜点、油炸食品
- ❌ 避免：含糖饮料、加工肉类

**饮水建议**
- 基础需求：体重(kg) × 35ml
- 运动后补充：每小时运动增加500-750ml
- 肾功能异常者需调整

### 🏃‍♀️ 运动建议

**心血管健康**
- 有氧运动：每周150分钟中等强度
- 具体活动：快走、游泳、骑车
- 强度控制：220-年龄 × 60-70%心率

**代谢健康**
- 阻力训练：每周2-3次
- 功能性训练：平衡和柔韧性
- 间歇训练：提高代谢效率

**个性化调整**
- 关节问题：避免高冲击运动，选择水中运动
- 心脏病史：严格心率监控，医生指导
- 久坐工作：每小时起立活动5分钟

### 💊 用药/补剂管理

**当前用药优化**
- 服药时间最佳化
- 药物相互作用监控
- 副作用管理策略

**营养补充建议**
基于体检结果：
- 维生素D不足：每日1000-2000IU
- 铁蛋白低：补铁+维生素C促吸收
- B12缺乏：口服或注射补充

**用药提醒系统**
- 固定时间用药
- 漏服补救措施
- 药物储存要求

### 🚨 就医信号识别

**立即就医**（24小时内）
基于个人病史的危险信号：
- 胸痛伴放射痛（有心血管风险者）
- 血糖 >16.7mmol/L（糖尿病患者）
- 血压 >180/110mmHg（高血压患者）

**尽快就医**（1周内）
- 持续3天发热 >38.5°C
- 体重短期内异常变化 >5kg
- 持续性异常症状

**定期复查信号**
- 血糖控制不佳的波动迹象
- 血压连续3天异常
- 用药后的预期反应时间

### 😴 睡眠优化

**基于监测数据**
- 睡眠时长目标：7-9小时
- 深睡眠占比 >20%
- 入睡时间规律性

**睡眠卫生**
- 固定作息时间
- 睡前1小时避免屏幕
- 卧室温度18-22°C

## 输出格式

### 📋 日常管理清单

**早晨（6:00-9:00）**
- [ ] 测血压（如有高血压）
- [ ] 服用晨服药物
- [ ] 早餐：燕麦片+牛奶+坚果
- [ ] 10分钟晨练或伸展

**上午（9:00-12:00）**
- [ ] 每小时起立活动5分钟
- [ ] 饮水500ml

**中午（12:00-14:00）**
- [ ] 午餐：蔬菜为主，适量蛋白质
- [ ] 餐后散步10-15分钟

**下午（14:00-18:00）**
- [ ] 下午茶时间选择无糖饮品
- [ ] 继续保持活动频率

**晚上（18:00-22:00）**
- [ ] 晚餐清淡，19点前完成
- [ ] 30分钟有氧运动
- [ ] 服用晚服药物
- [ ] 21:30开始睡前准备

### 📱 监测要点

**日常自测**
- 血压：每日固定时间（如有高血压）
- 血糖：餐前餐后（如有糖尿病）
- 体重：每周同一时间

**症状记录**
- 异常症状出现时间、程度、持续时间
- 可能诱因分析
- 缓解措施效果

## 家人简化版

当需要给家人的版本时，精简为5-8条最重要的要点：

**简化原则**
- 只保留最关键的行为改变
- 用具体行动替代抽象概念
- 每条配一个明确的执行标准

**示例**
1. 每天早上8点测血压，记录在本子上
2. 早餐不吃油条包子，改成燕麦片加鸡蛋
3. 每天饭后散步30分钟，不下雨就出门
4. 晚上10点前睡觉，手机放客厅充电
5. 药物放在床头，每天早上7点服用
