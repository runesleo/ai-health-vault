# 复查日历生成

基于家庭成员的健康档案，生成个性化的复查提醒日历，确保不错过重要的健康检查。

## 核心任务

分析每个家庭成员的健康档案和就医记录，生成结构化的复查日历，并可导出为标准日历格式。

## 分析维度

### 1. 复查项目识别

**常规复查**
- 年度体检（每12个月）
- 血常规、生化全套（根据年龄调整频率）
- 专项筛查（如妇科检查、前列腺检查）

**疾病相关复查**
- 慢性病监测（糖尿病、高血压等）
- 术后复查（按医嘱时间表）
- 异常指标跟踪复查

**预防性检查**
- 癌症筛查（按年龄和风险因素）
- 疫苗接种（如流感疫苗、新冠疫苗）
- 眼科、口腔科定期检查

### 2. 紧急程度分级

**🔴 已过期**
- 超过建议复查时间
- 标注过期天数
- 优先安排

**⚠️ 即将到期**（1个月内）
- 需要开始预约
- 提前提醒准备

**✅ 未到期**
- 正常监控状态
- 设置提前提醒

### 优先级排序
按紧急程度和健康重要性排序，最需要关注的放在最前面。

## 个性化调整

### 异常指标加速复查
- 上次检查异常→缩短复查间隔
- 如血糖异常从年检改为3-6个月
- 肿瘤标志物异常→3个月复查

### 慢性病管理
**糖尿病患者**
- 血糖：每3个月
- HbA1c：每3-6个月
- 眼底检查：每年
- 足部检查：每年

**高血压患者**
- 血压监测：每1-3个月
- 心电图：每6-12个月
- 肾功能：每6-12个月

### 年龄相关调整
**40岁以上**
- 增加肿瘤筛查频率
- 心血管风险评估
- 骨密度检查

**65岁以上**
- 认知功能评估
- 跌倒风险评估
- 更频繁的全面体检

## 日历导出功能

### ICS 格式生成
```
BEGIN:VCALENDAR
VERSION:2.0
PRODID:Health Checkup Calendar

BEGIN:VEVENT
SUMMARY:妈妈血糖复查提醒
DTSTART:20260415T090000
DTEND:20260415T100000
DESCRIPTION:建议到内分泌科复查血糖，上次结果偏高
LOCATION:XX医院内分泌科
BEGIN:VALARM
TRIGGER:-P7D
DESCRIPTION:血糖复查提醒（提前7天）
END:VALARM
END:VEVENT

END:VCALENDAR
```

### 提醒设置
- **主提醒**：复查当天
- **预提醒1**：提前1周（准备预约）
- **预提醒2**：提前1天（确认准备）

## 智能建议

### 批量安排优化
- 同一家庭成员的多项检查合并安排
- 同一医院的检查集中在一天
- 考虑工作日程安排最佳时间

### 季节性考虑
- 疫苗接种的最佳时机
- 体检淡旺季安排
- 节假日医院排班影响

### 费用优化
- 医保报销周期考虑
- 体检套餐优惠时机
- 家庭成员团体优惠

## 维护更新

### 自动更新触发
- 完成复查后更新下次时间
- 新增疾病诊断后调整频率
- 医生修改复查建议后同步

### 历史记录
- 保留复查履行记录
- 跟踪延迟复查的原因
- 分析复查依从性趋势

## 特殊情况处理

### 紧急插入复查
- 症状变化需要额外检查
- 体检发现异常需要专科复查
- 用药调整后的监测

### 复查结果异常
- 自动生成下次复查计划
- 调整复查频率
- 设置专科跟踪

## 触发场景

- 用户询问"什么时候该复查了"
- 家庭成员体检后更新复查计划
- 用户要求生成复查日历
- 需要导出到手机日历应用
- 定期（如每月）检查复查提醒
