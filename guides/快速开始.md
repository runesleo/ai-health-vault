# 快速开始：30 分钟搭建家庭健康管理系统

## 你需要准备的

- [Obsidian](https://obsidian.md/)（免费，全平台）
- 任一 AI 工具：Claude / ChatGPT / Gemini（免费额度就够）
- 家人的体检报告（纸质拍照 or 电子版）

## Step 1: 获取模板（2 分钟）

```bash
# 方式一：Git Clone
git clone https://github.com/runesleo/ai-health-vault.git

# 方式二：下载 ZIP
# 点击 GitHub 页面右上角 Code → Download ZIP
```

## Step 2: 用 Obsidian 打开（1 分钟）

1. 打开 Obsidian
2. 点击「打开文件夹作为仓库」
3. 选择 `ai-health-vault/vault/` 文件夹
4. 你会看到健康管理中心作为首页

## Step 3: 填写家庭成员信息（5 分钟）

1. 打开 `健康管理中心.md`
2. 把 `[你的名字]`、`[家人1]` 等替换成真实名字或代号
3. 打开 `家庭成员健康档案.md`，填入每人的基本信息

## Step 4: 创建个人体检档案（2 分钟）

1. 复制 `成员模板/体检档案-模板.md`
2. 改名为 `体检档案-[名字].md`（比如 `体检档案-爸爸.md`）
3. 每个家庭成员一份

## Step 5: 录入第一份体检报告（10 分钟）

1. 拍照或上传体检报告
2. 打开 `prompts/01-体检报告提取.md`，复制里面的 Prompt
3. 把 Prompt + 照片一起发给 AI
4. AI 输出结构化表格 → 粘贴到对应的体检档案

## Step 6: 导入 Apple Watch 数据（可选，10 分钟）

1. 按 `prompts/05-Apple-Watch数据分析.md` 的步骤导出数据
2. 让 AI 分析 → 结果写入体检档案的 Apple Watch 区域

## 搭建完成！

以后的日常使用：

| 场景 | 操作 |
|------|------|
| 新体检报告 | 拍照 → AI 提取 → 粘贴到档案 |
| 买了新药 | 拍药盒 → AI 识别 → 更新用药记录 |
| 要去看病 | 告诉 AI 看什么科 → 生成就医准备清单 |
| 给父母解释 | AI 分析结果 → 转成微信口语版 |
| 季度复盘 | 导出 Apple Watch 数据 → AI 趋势分析 |

## 常见问题

### AI 识别体检报告不准确？

- 确保拍照清晰、光线充足
- 多页报告分开拍，每页一张
- 识别后核对关键数字（血糖、血脂等）

### Gemini / ChatGPT 处理不了大文件？

Apple Watch 导出的 XML 可能超过 1GB。解决：
1. 先运行仓库内脚本：`python scripts/apple_health_preprocess.py --input /path/to/export.zip --output ./apple-health-csv`
2. 按数据类型（心率、步数、睡眠）拆分成独立文件
3. 分批喂给 AI 分析
4. 如果当前环境没法运行脚本，再让 AI 临时写一个兜底脚本

详见 `prompts/05-Apple-Watch数据分析.md`。

### 数据安全？

- 所有数据存在你自己的电脑上（Obsidian 是本地软件）
- 发给 AI 分析时，数据会经过 AI 服务商的服务器
- 如果极度敏感，可以用本地模型（如 Ollama）替代云端 AI
- 建议不要在档案里存身份证号等敏感信息
