## 工作流

### 收到新体检报告时

```
1. 拍照/PDF 发给 AI
2. AI 提取数据 → 写入对应的「体检档案-XX.md」
3. 原件存入 → 健康管理/XX/报告原件/
4. AI 自动对比历史数据 → 标注趋势变化
5. 如有异常 → 生成复查追问清单
```

### 买了新药/保健品时

```
1. 拍药盒照片 → 发给 AI
2. AI 识别药名、剂量、频次 → 写入用药记录
3. AI 检查是否与现有用药冲突
```

### 复查/就医前

```
1. 告诉 AI "我要去看 XX 科"
2. AI 读取历史档案 → 生成：
   - 历史趋势摘要（带给医生看）
   - 本次应关注的指标
   - 建议追问医生的问题
3. 就医后 → 把医生的结论发给 AI → 更新档案
```

### 导入 Apple Watch / 智能手环数据

```
1. iPhone 健康 App → 右上角头像 → 拉到最底部 → 导出所有健康数据
2. 得到 export.zip（几十 MB ~ 几百 MB）
3. 传到电脑，告诉 AI 文件路径
4. AI 解析 XML → 生成心率/步数/睡眠/运动分析报告
5. 关键指标写入体检档案
```

> 建议每季度导出一次，看趋势变化。
